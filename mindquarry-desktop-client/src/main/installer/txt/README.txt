====================================================================
 INTRODUCTION
====================================================================
The Mindquarry Desktop Client allows working with Mindquarry 
Collaboration Server instances in a user-friendly manner and without
a web browser.

It allows you to manage/create tasks and to synchronize your local 
documents with the server and keeps track of changes, that are made
to the documents.

====================================================================
 NEW FEATURES AND FUNCTIONS IN THIS RELEASE 
====================================================================
This release implements these features and functionality:

1. Creation of one or more server profiles
2. Synchronizing local workspaces and documents with the server
3. Listing of tasks that are stored on the server
4. Editing of tasks and update them on the server
5. Creation of new tasks and save on the server

For a complete usage description please have a look at the user 
guide which is available at http://www.mindquarry.com/community

====================================================================
 KNOWN LIMITATIONS AND NOTES
====================================================================
These limitations are known to exists in this release of the software:

Limitation: Creating/Editing tasks in offline mode
Workaround: Only edit/create tasks when you are connected to the 
			internet

Limitation: Conflict resolving is not supported
Workaround: If a file is in conflict, you will be ask which version 
			should be treated as the new version.
			
Limitation: Document meta data is still stored in .svn folders
Workaround: This is not really a problem, but be careful!!!
			DO NOT delete .svn folders within your workspace 
			directories!!!
